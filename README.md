# CMS Starter Template

Collection of UI components and library of utility classes. To get started, move the desired content from 'Library' directories into 'Component' directories for quick setup. Be sure to delete any unwanted library content.

## Components

- Accordion
- Carousel Variants
- Counter
- Form Variants
- Marquee
- Tabs
- Video Overlay

## Section Components

- Carousel Hero
- Price Cards
- Scroll Horizontal
- Scroll Stack

## Utilities

- Cursor Target
- Fill Text
- Glitch Text
- Gradient Border
- Grid Fade
- Magnet
- Parallax
- Tilt Cards
- Typing Text
